# SPOR
I 1800-tallet udgav det danske politi de såkaldte "Politiets Efterretninger", som, bredt forstået, indeholder relevant information om kriminel aktivitet, der skulle sendes rundt til politiafdelinger henover Danmarks areal. Det er i forbindelse med et kursus på Aalborg Universitet blevet digitaliseret og tilgængeliggjort. Her er dokumentationen.
